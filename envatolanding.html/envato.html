Key Features

Popup Delay Control

Set a custom time delay for the popup to appear, grabbing attention at the perfect moment.

WooCommerce Coupons

Automatically generate and display unique WooCommerce coupon codes for each visitor.

Email Capture

Grow your mailing list by requiring an email address before revealing the coupon code.

Celebration Animation

Engage users with a fun confetti animation when they successfully claim a coupon.

One-Click Copy

A convenient "Copy" button allows users to easily copy the coupon code to their clipboard.

Multilingual Ready

Fully compatible with WPML and Polylang, making your popups ready for a global audience.

Installation Guide

  1. Download: After purchasing, download the plugin ZIP file from your Envato account.

  2. Upload: Navigate to your WordPress Dashboard, go to Plugins > Add New, and click "Upload Plugin".

  3. Install & Activate: Choose the downloaded ZIP file, click "Install Now", and then "Activate Plugin".

  4. Configure: Find the "Smart Coupon Popup" menu in your dashboard to customize settings and styles.
